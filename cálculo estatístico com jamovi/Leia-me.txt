Este repositório contém os resultados estatísticos das variáveis testadas em um experimento.
Os cálculos estatísticos foram realizados utilizando o software Jamovi, que oferece recursos
 de estatística descritiva para análise de dados.

Resultados Estatísticos
Nesta pasta, você encontrará os resultados estatísticos obtidos para cada variável testada no experimento. 
Utilizando o Jamovi, foram aplicadas técnicas de estatística descritiva, incluindo as seguintes medidas: 
média, mediana, desvio-padrão, variância, amplitude, mínimo, máximo, assimetria, erro padrão, curtose, 
erro padrão da curtose, W de Shapiro-Wilk e p Shapiro-Wilk.

Teste t para Amostras Independentes
O teste t para amostras independentes foi utilizado neste experimento, utilizando a correção de Welch. 
Em todos os testes realizados, foi adotado um nível de significância de p < 0,05.

Observações
